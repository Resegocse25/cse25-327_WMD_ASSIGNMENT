About Routines & Habits

This website was created as part of a Computer Systems Engineering (CSE) web development assessment. The purpose of the project is to design and build a static prototype website using HTML, and later apply CSS and other web development principles as they are introduced during the course.

Why This Topic Was Chosen

The topic “Routines and Habits” was chosen because time management and effective studying are common challenges faced by many students. As a student who has personally struggled with procrastination, inconsistency, and retaining information, this topic felt relevant and meaningful.

Rather than presenting expert advice, this website explores general ideas and study techniques that students can experiment with and adapt to their own needs.

Project Goals

The main goals of this project are to:

    Practice building structured web pages using HTML
    Apply semantic elements such as headers, navigation, main content, and footers
    Organise content clearly across multiple pages
    Gradually improve the design using CSS as new principles are learned

This website is not intended to be a fully functional online platform, but a prototype that demonstrates understanding of web development fundamentals.

Technologies Used

  • GitHub for version control and hostingCSS (to be implemented and improved over time)

    Project Development Approach

    The website is being developed step by step. Content and structure were planned first before styling was introduced. As new concepts are taught during lectures and tutorials, the website will be updated and refined.

    This gradual approach reflects the learning process and ensures that each principle is properly understood before being applied.

    This project represents both a technical learning experience and a personal exploration of routines, habits, and time management strategies. The website will continue to evolve as skills in web development improve.
